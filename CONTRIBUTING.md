# Contributing to SossoClicker

Thank you for your interest in SossoClicker!  

Before you contribute or create a modified version, please read these rules:

## Rules
1. You must ask EllTeam before uploading, redistributing, or publishing any modified version of SossoClicker.
2. Contributions are welcome, but the official repository and releases remain under Elias’ control.
3. Always give proper credit to the original creator (Elias).

Following these rules helps keep the project organized and prevents unapproved versions from being published.
