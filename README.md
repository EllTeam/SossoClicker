# SossoClicker

SossoClicker is a simple clicker game made as a fun project.

## About

Click to gain points and keep increasing your score.
This project is created mainly for learning and experimentation.

## How to Play

1. Open the game in your browser.
2. Click the Sosso image to earn points.
3. Try to get the highest score possible!

## Running Locally

If you want to run the game locally:

1. Start a local server:

   ```
   python -m http.server 8000
   ```
2. Open your browser and go to:

   ```
   http://localhost:8000
   ```
